# snoopy-le-sabe
Le sabe

## Description
This project is a PoC of an autonomously driven car, for it we will design a physical UI to see the car's telemetry and the objective is for 
1. Succesfully working screen in car
2. Real statistics shown (revs, pwm, distance)
3. Real movements
Those three objectives are the progressive milestones

## Implementation
The implementation of the project will be with an FPGA colorlight 5a-75e, a motor along with its driver, a 64x64 led matrix, a hall sensor, a lidar, and a temperature sensor. (this might change)
